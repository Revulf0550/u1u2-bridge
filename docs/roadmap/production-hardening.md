# Production hardening

## Статус
**ПОСЛЕ полевых** — основной набор. **Hardware watchdog — кандидат сделать РАНЬШЕ**
(надёжность — приоритет №1), решение пока не принято.

## Суть
Закалка устройств для боевой эксплуатации:

- **RO-rootfs (overlayroot)** — защита ФС от повреждения при внезапном отключении питания.
- **Логи на tmpfs** — чтобы RO-rootfs не мешал записи и не убивал ресурс SD/NVMe.
- **OSD RSSI/SNR на видео** — наложение статистики линка на картинку для оператора.

### Hardware watchdog — отдельно, кандидат на «раньше»
- `rockchip-wdt` (`/dev/watchdog0`), systemd `RuntimeWatchdogSec=20`.
- Автоперезагрузка при зависании — прямой вклад в надёжность (приоритет №1).
- **Решение, делать ли раньше остального hardening, пока не принято.**

## Когда
Основной набор (RO-rootfs, tmpfs-логи, OSD) — после полевых испытаний.
Hardware watchdog — возможно раньше, по отдельному решению.
